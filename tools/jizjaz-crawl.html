Security Suite v2.0

JIZJAZ-CRAWL

Alat pengikis data (scraper) performa tinggi untuk mengidentifikasi informasi sensitif seperti kredensial, API key, dan kebocoran konfigurasi.

01 Penggunaan Cepat

Jalankan skrip dengan menyediakan daftar URL target. Alat ini akan memproses setiap URL untuk mencari pola data sensitif.

$ ./jizjaz-crawl.sh targets.txt

02 Logika Teknis

Global Search Engine

Menerapkan strategi pencarian global non-greedy. Mengekstrak setiap kemunculan unik kata kunci dalam konten URL target, memastikan tidak ada kredensial yang tersembunyi dalam file besar yang diminimalkan.

Contextual Offset

Mengambil buffer 40 karakter di sekitar setiap temuan. Memungkinkan analis untuk melihat deklarasi variabel lengkap atau pasangan key-value JSON yang terkait dengan temuan tersebut.
