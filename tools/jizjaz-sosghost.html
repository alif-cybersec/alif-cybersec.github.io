Security Suite v2.6

JIZJAZ-SOSGHOST

Alat pendeteksi Broken Social Media Link (BSML) yang canggih dengan fitur Stealth Mode dan Deep Inspection untuk mengidentifikasi potensi pembajakan akun (hijacking).

01 Penggunaan Cepat

Jalankan skrip dengan memberikan URL target. Alat ini akan merayapi halaman untuk mencari link media sosial dan memverifikasi status ketersediaan akun tersebut.

$ python3 jizjaz_sosghost.py -u https://target-website.com

02 Fitur Utama

Deep Inspection (Soft 404 Detection)

Tidak hanya mengandalkan kode status HTTP, alat ini menganalisis konten halaman (Title & Keywords) untuk mendeteksi profil yang sudah dihapus namun tetap memberikan respons kode status 200 (Soft 404).

Stealth Mode (Evasion)

Menggunakan delay acak (1.5s - 3.5s) dan browser headers yang menyerupai perilaku manusia untuk menghindari deteksi anti-bot pada platform media sosial besar.

Multi-Platform Support

Mendukung pemindaian otomatis untuk berbagai platform populer seperti Instagram, Facebook, GitHub, TikTok, LinkedIn, YouTube, Discord, dan Telegram.
